# 🚆 RailEase

RailEase is a modern railway management and train information desktop application built using **C# .NET Windows Forms**. It provides a clean and user-friendly interface for searching trains, checking PNR status, tracking live train locations, viewing seat availability, and managing bookings.

## ✨ Features

- 🏠 Dashboard
- 🔍 Search Trains
- 🎫 PNR Status
- 🚆 Live Train Status
- 💺 Seat Availability
- 💰 Fare Enquiry
- 📖 My Bookings
- ❤️ Favorites
- 📍 Station Information
- 👤 User Profile
- ⚙️ Settings

## 🎨 UI

- Modern dark theme
- Professional sidebar navigation
- Interactive search panel
- Responsive dashboard cards
- Railway-inspired design

## 🛠️ Built With

- C#
- .NET 8
- Windows Forms
- Visual Studio
